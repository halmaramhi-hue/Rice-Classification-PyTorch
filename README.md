# Rice Classification using PyTorch

A deep learning project for rice image classification using PyTorch. This project includes data preprocessing, model training, evaluation, and prediction on a rice classification dataset.

## Features
- Data preprocessing
- Deep learning model built with PyTorch
- Model training and validation
- Performance evaluation
- Rice image classification

## Technologies
- Python
- PyTorch
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Dataset
Rice Classification Dataset (.csv)
